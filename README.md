## üìù Project Description

**WiseOwlEnglish** is built to create a "learn through play" environment for primary students. The project focuses on:
- **Vocabulary through Images:** Colorful flashcards covering topics like animals, colors, and family.
- **Pronunciation Practice:** Integrated listening features and basic voice recognition.
- **Interactive Games:** Engaging mini-games such as matching pictures with words, filling in the blanks, and listening to pronunciations to choose the correct answer, all designed to effectively reinforce knowledge.
- **Progress Tracking:** Helping parents monitor how many lessons their children have completed.
